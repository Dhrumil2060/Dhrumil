Network Penetration Testing

This project involved conducting a comprehensive network penetration test to identify and exploit vulnerabilities within the target environment.

Tools Used: Nmap, Nessus, Metasploit, Wireshark

Key Achievements:

  • Performed network scanning and vulnerability assessment.
  • Executed exploit attempts to verify security weaknesses.
  • Documented findings and provided remediation recommendations.
  • Enhanced understanding of attack vectors and network defense.
