Virtualization Lab (VMware ESXi)

Configured a complete virtual lab using VMware ESXi 6.7 and vCenter Server Appliance (VCSA). Implemented networking with vSwitches, created and managed virtual machines, enabled DRS (Distributed Resource Scheduler) and HA (High Availability), and configured VM backups and snapshots. The lab also included simulation of scalable enterprise-like environments for testing IT infrastructure scenarios.
