Virtualization and Vulnerability Assessment

This project involved setting up a virtual environment and performing end-to-end vulnerability assessment. It integrated modern cybersecurity practices across various stages of the system lifecycle.

  • Tools Used: Oracle VirtualBox, Windows 10 ISO, Kali Linux ISO, VM network bridging
  • Key Tasks:
    • Installed and configured two VMs with custom specs
    • Set up shared folders and guest additions
    • Implemented bridged networking to allow internet access
    • Captured network settings and screenshots of configurations
    • Verified system connectivity using ping, IP configuration, and VirtualBox settings
  • Skills Demonstrated: VM provisioning, OS installation, networking configuration, system hardening basics
