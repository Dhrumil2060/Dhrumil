Network Penetration Testing

Executed a full penetration testing lifecycle using tools like Nmap, Wireshark, Hashcat, John the Ripper, and Responder to identify and exploit vulnerabilities in simulated networks.

Key Techniques:

  • Nmap: SYN, TCP, UDP, XMAS, and RPC scanning
  • Wireshark: Live traffic inspection of DNS/ICMP/HTTP/FTP packets
  • Hashcat: Dictionary + combinator attacks on captured NTLM/MD5 hashes
  • Responder: LLMNR/NBT-NS spoofing and multi-relay simulations
  • Reporting: Screenshots, tool output, and mitigation aligned to OWASP Top 10

Outcome: Identified high-risk vulnerabilities, extracted valid credentials, and validated network security posture through real-world attack sce
