NIST-CSF Strategy Implementation

This project focused on the practical implementation of the NIST Cybersecurity Framework (CSF) in a simulated small-to-mid-sized enterprise environment. The goal was to assess the current cybersecurity posture, identify gaps, and design a custom roadmap for maturity improvement.

  • Mapped current controls to the NIST-CSF functions: Identify, Protect, Detect, Respond, Recover.
  • Conducted gap analysis using the CSF Implementation Tiers.
  • Designed a control roadmap tailored to compliance and business objectives.
  • Created visuals in Power BI to communicate control coverage and priority areas.
  • Recommended cloud security integration and endpoint visibility as top priorities.
