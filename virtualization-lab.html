Virtualization Lab

Designed a secure enterprise virtual lab using VMware ESXi and vCenter. Simulated endpoint threats, malware, and recovery strategies using snapshot rollbacks and isolated network zones.

Key Components:

  • VMware ESXi: Bare-metal hypervisor for virtualized test environments
  • vCenter Server: Centralized management of VMs and clusters
  • Security Zones: Configured VLANs and firewall rules for DMZ/intranet isolation
  • Endpoint Simulation: Loaded Kali Linux, Windows Server, and custom agents for real-world threat scenarios

Outcome: Built a flexible security testing platform to experiment with malware behavior, SIEM logs, and rapid response workflows in a controlled, safe lab.
